# 手机端新闻前台页面+后台上传信息页面PHP版
这是用PHP搭建的服务器端页面，包含前台页面和后台上传数据信息的页面
# 网站技能以及实现的功能
技术主要有前端：HTML/css/js/jquery/ajax<br><br>
后端：php/mysql<br><br>
功能：在后台页面添加新闻标题等信息，可在前台自动生成新闻展示界面
